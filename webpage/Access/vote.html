What is voting?

This voting feature allows you to choose what world you want to run on the server on a daily basis. The server tallies the votes at 1:00 every day. Members can only vote once per day. The specific SMPs running on the server will gradually change, however the world numbers you vote for will have the same kind of SMPs: one for vanilla, one for modded, and one for events.

Available Worlds

Name Description
Data (Vanilla) This container runs the current vanilla SMP. Vanilla means normal minecraft without modifications.
Data1 (Modded) This container runs the current modded SMP. Modded means minecraft with mods.
Data2 (Events) This container runs events and is used for testing. Sometimes it might not be running anything. Check the discord server to see if there is an event running.
