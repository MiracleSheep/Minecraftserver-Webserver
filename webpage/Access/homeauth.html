Welcome!

This is a website created for the MiracleSheep Minecraft server and Discord server! It contains many memories of the server. It also has a few other secret features only accessible by members of the server. We do all sorts of Minecraft activities, such as hardcore group speedruns, vanilla and modded survival worlds, as well as adventure maps. Enjoy exploring our website!

How do I become a server member?

To become a server member, You need to know MiracleSheep or another member of the server. All you have to do is ask, and then you are added to both the minecraft server and the discord. Ask MiracleSheep about having an account on this website.
