Plugins

Why plugins are here

The owner of the server, MiracleSheep, has made a few minecraft plugins, all of which ran on the server at some point. This page is where people can find the plugins. Some of these plugins are not very well coded. They have many bugs because MiracleSheep coded them before he knew programming conventions (and he is too lazy to fix them now). There will be more plugins that come out, so check this page every once in a while to see if any new ones pop up. The links here will direct you to the github repository where you can download the plugins under the releases section.

Duel Plugin

This is both the first and one of the biggest plugins MiracleSheep ever made (not a good idea to start with a big project). As a result, it's very buggy and has lots of difficulty with more than two players. This plugin is only here for history, not for use. It has a custom kit system, which allows players to create kits in the config and equip them if they have certain permissions. It also allows the user to choose what world and coordinates they want the duel arena to be in, and allows the server owner to make their own arena. The duelists can choose what kit they want, and each kit can have their own arena assigned.

HardCore Revival Plugin

This plugin is very simple. All it does is provide a craftable crystal that can revive a player in hardcore. The crystal requires 1 diamond and eight iron to craft. It was used because people died too often during hardcore events. One compatibility issue to look our for: the multiverse plugin has an annoying feature which swiches the player's gamemode to the dimension's default when they change dimensions This interferes with the revival crystal, as it only picks up players in spectator mode. To get around this, set the default multiverse gamemode in the overworld to spectator.
