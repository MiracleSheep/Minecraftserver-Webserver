Plugins

There have been many custom made plugins that have run on the server. This page is where people can find the plugins. The plugins are mainly made by MiracleSheep, but others are welcome to contribute their plugins to be added to this webpage. Some of these plugins are not very well coded. The plugins here are not meant to be perfect, this page is simply documenting which plugins have run on the server. The website will mention whether or not each plugin is well made in the description. There will be more plugins that come out, so check this page every once in a while to see if any new ones pop up. The links here will direct you to the github repository where you can download the plugins under the releases section.

Duel Plugin - MiracleSheep

TThis is both the first and one of the biggest plugins MiracleSheep ever made (not a good idea to start with a big project). As a result, it's very buggy and has lots of difficulty with more than two players. It was used temporairily in the second vanilla SMP and later removed due to the bugs. This plugin is only here for history, not for use. It works by allowing players to duel with /duel and permitting them to select a kit before teleporting them to an arena in another world. It depends on the multiverse plugin. It has a custom kit system, which allows players to create kits in the config and equip them if they have certain permissions. It also allows the user to choose what world and coordinates they want the duel arena to be in, and allows the server owner to make their own arena. The duelists can choose what kit they want, and each kit can have their own arena assigned. The bugs occur when there are multiple duels occurring at once. Sometimes it works, sometimes it doesn't. Essentials Plugin - MiracleSheep

This plugin is well coded and has a variety of features. It was used in the second Vanilla SMP as a helper/regulatory plugin. It contains a save inventory feature, multiplayer sleep feature, message upon player joining and a /rules command, just to name a few. Check the config to discover all the other features, which can all be toggled on and off. Here is the download:

Hard-Core-Revival Plugin - MiracleSheep

This plugin is well coded and very simple. All it does is provide a craftable crystal that can revive a player in hardcore. The crystal requires 1 diamond and eight iron to craft. The plugin comes with a custom texture pack made by Crazyfr0g_. It was used because people died too often during hardcore events. One compatibility issue to look out for: the multiverse plugin has an annoying feature which swiches the player's gamemode to the dimension's default when they change dimensions. This interferes with the revival crystal, as it only picks up players in spectator mode. To get around this, set the default multiverse gamemode in the overworld to spectator. Here is the download:

Youtube Minigame Plugin - MiracleSheep

This is a well coded plugin, inspired by the minigames that the youtuber dream plays. This plugin contains three of the games Dream played on his channel; Block Hunt, Death Swap, and ManHunt. It also features a handy queue system for big minigames. One problem with this plugin is that the games are a lot harder than how youtubers seem to portray it. To get around this, simply change the settings of each minigame to your liking through the config. The plugin is meant to be played in version 1.17 and onward. Here is the download:
